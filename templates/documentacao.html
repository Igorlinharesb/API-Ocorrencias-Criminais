Base de dados por Municipio


O usuário tem a opção de acessar todos os registros da base de dados por município através da requisição HTTP abaixo:

GET http://3.86.8.77/municipio/page=1 HTTP/ 1.1

Serão retornados os registros de todos os municípios da base de dados


O usuário tem a opção de acessar todos os registros da base de dados por município de um estado específico passando o parâmetro uf, conforme o exemplo de requisição HTTP abaixo:

GET http://3.86.8.77/municipio/uf=AC/page=1 HTTP/1.1

Serão retornados os registros de todos os municípios do estado passado como parâmetro, no caso do exemplo acima, todos os municípios do estado do Acre.


O usuário tem a opção de acessar todos os registros da base de dados por município em um intervalo de datas (no formato mês-ano) passando os parâmetros início e fim conforme o exemplo de requisição HTTP abaixo:

GET http://3.86.8.77/estado/inicio=01-19&fim=12-19/page=1 HTTP/1.1

Serão retornados os registros de todos os municípios entre a data passada no parâmetro início e a data passada no parâmetro fim (no caso do exemplo acima, entre janeiro e dezembro de 2019).


O usuário também pode passar apenas um dos parâmetros acima. Caso seja passado apenas o parâmetro inicio será retornado todos os registros de todos os municípios entre a data passada no parâmetro inicio e a data do dado mais recente. Caso seja passado apenas o parâmetro fim, serão retornados os registros de todos os municípios entre a data do registro mais antigo e data passada no parâmetro fim. A seguir temos 2 exemplos:

GET http://3.86.8.77/municipio/inicio=01-19/page=1 HTTP/1.1

GET http://3.86.8.77/municipio/fim=12-19/page=1 HTTP/1.1

No primeiro exemplo serão retornados todos os registros a partir de janeiro de 2019, enquanto no segundo exemplo serão retornados todos os registros até dezembro de 2019.


Por fim, o usuário pode combinar as buscas anteriores, buscando por um estado específico, dentro de um intervalo de datas conforme a requisição HTTP abaixo:

GET http://3.86.8.77/municipio/uf=AC/inicio=01-19&fim=12-19/page=1 HTTP/1.1

Assim, como as requisições por data, também é possível passar apenas um dos parâmetros, ou só o parâmetro inicio ou só o parâmetro fim.

GET http://3.86.8.77/municipio/uf=AC/inicio=01-19/page=1 HTTP/1.1

GET http://3.86.8.77/municipio/uf=AC/fim=12-19/page=1 HTTP/1.1

Nesses últimos casos, são retornados todos os registros de um estado específico no intervalo desejado pelo usuário


Base de Dados por Estado
O usuário tem a opção de acessar todos os registros da base de dados por estado através da requisição HTTP abaixo:

GET http://3.86.8.77/estado/page=1 HTTP/ 1.1

Serão retornados os registros de todos os municípios da base de dados


O usuário tem a opção de acessar todos os registros da base de dados de um estado específico passando o parâmetro uf conforme o exemplo de requisição HTTP abaixo:

GET http://3.86.8.77/estado/uf=AC/page=1 HTTP/1.1

Serão retornados os registros do estado passado como parâmetro, no caso do exemplo acima, do estado do Acre.


O usuário tem a opção de acessar todos os registros da base de dados por estado em um intervalo de datas (no formato mês-ano) passando o parâmetro inicio e o parametro fim conforme o exemplo de requisição HTTP abaixo:

GET http://3.86.8.77/estado/inicio=01-19&fim=12-19/page=1 HTTP/1.1

Aqui, serão retornados os registros de todos os estados entre a data passada no parâmetro inicio e a data passada no parâmetro fim, (no caso do exemplo acima, entre janeiro e dezembro de 2019).


O usuário também pode passar apenas um dos parâmetros acima. Caso seja passado apenas o parâmetro inicio serão retornados os registros de todos os estados entre a data passada em no parâmetro inicio e a data do registro mais recente. Caso seja passado apenas o parâmetro fim, serão retornados os registros de todos os estados entre a data do registro mais antigo e data passada no parâmetro fim. Abaixo temos 2 exemplos:

GET http://3.86.8.77/estado/inicio=01-19/page=1 HTTP/1.1

GET http://3.86.8.77/estado/fim=12-19/page=1 HTTP/1.1



Por fim o usuário pode combinar as buscas anteriores, buscando por um estado específico dentro de um intervalo de datas, conforme a requisição HTTP abaixo.

GET http://3.86.8.77/estado/uf=AC/inicio=01-19&fim=12-19/page=1 HTTP/1.1

Assim como as requisições por data também é possível passar apenas um dos parâmetros, ou só inicio ou fim.

GET http://3.86.8.77/estado/uf=AC/inicio=01-19/page=1 HTTP/1.1

GET http://3.86.8.77/estado/uf=AC/fim=12-19/page=1 HTTP/1.1

Nesses últimos casos, são retornados todos os registros de um estado específico, no intervalo desejado pelo usuário.
